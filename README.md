# 📖 BookClubApp - Personal Book Tracker with AI Search

A personal reading diary to save, rate, and reflect on books -
with an AI-powered assistant to help you find your next read.

> 🚧 **Status: In Development**

---

## 💡 Idea & Motivation

A personal library manager meets a reading community - users can save their books,
join clubs, share reviews, and find their next read using AI-assisted search.

---

## ✨ Planned Features

### 📖 Personal Library

- Save books to personal reading list (want to read / reading / finished)
- Rate and review books
- Fetch book details automatically via API

### 👥 Community

- Create or join book clubs
- See what other members are reading

### 🤖 AI-Powered Search

- Natural language search: "I want something like Harry Potter but darker"
- Book recommendations based on reading history
- Powered by OpenAI API

### 🔐 User Auth

- Registration & login
- Personal profile with reading stats

---

## 🛠️ Planned Tech Stack

### Backend

### Frontend

- HTML / CSS

### Tools

- Git, GitHub
- Postman (API testing)
- pytest (unit tests)

## 🗄️ Planned Database Structure

| Table   | Description                          |
|---------|--------------------------------------|
| users   | User accounts and profiles           |
| books   | Book data (title, author, cover)     |
| authors | Author data (name, birth/death date) |
| clubs   | Book clubs (name)                    |
